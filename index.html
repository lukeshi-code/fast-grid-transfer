Fast Grid Transfer

Transfer files through a high-density colored screen grid. The encoder displays binary frames; the decoder captures the screen and decodes frames in a Web Worker.

Open Encoder Open Decoder

Colored Grid Encoding

Each frame stores 1024 raw bytes using 2-bit color cells, with binary headers and per-frame checksums.

Screen Capture Decoder

The decoder reads the encoder tab or window directly from browser screen capture.

Worker-Based Decode

Frame detection and sampling happen off the main thread to keep the UI responsive.
