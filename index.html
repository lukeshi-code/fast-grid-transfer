Fast Grid Transfer

Start here for every workflow. Use Encoder and Decoder for normal file transfer, or Delta Packager for code directories; the delta workflow opens Encoder automatically after it builds the archive.

Code Delta Transfer Manual Encoder Decoder

One Startup File

Run `run-fast-grid.bat` once and keep that window open. All pages and local APIs come from that server.

Code Delta Transfer

Pick a Git or SVN working copy, review changed files, build a TAR archive, then auto-load it in Encoder.

Manual File Transfer

Open Encoder manually for a file or folder, then capture that tab from Decoder on the receiving side.

Verified Output

The receiver rebuilds the stream and verifies SHA-256 before downloading the final file.
